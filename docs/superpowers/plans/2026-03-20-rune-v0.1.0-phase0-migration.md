# Rune v0.1.0 Phase 0: Repository Migration

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate POC v4 code from `demo/poc-v4/` to the official monorepo structure at the repository root, with `runtime/`, `sdks/`, `proto/` layout.

**Architecture:** Copy (not move) POC v4 Rust crates into `runtime/`, proto into `proto/`, create root workspace Cargo.toml. Update all path dependencies to reflect the new `runtime/` prefix. Proto build.rs path changes from `../proto/` to `../../proto/`. SDK and example directories are created as placeholders.

**Tech Stack:** Rust (Cargo workspace), protobuf, tonic-build

**Spec:** `docs/scope/v0.1.0.md` — Section "Repository Structure"
